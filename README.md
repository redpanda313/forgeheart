# ForgeHeart: Gift of the Brass Gods

Heartfelt steampunk — wake your brother’s soul in a brass frame, then build a sky empire of markets, workshops, and living cities.

> Separate product from [Trump Doom](https://github.com/redpanda313/trump-doom).

## Run locally

```bash
git clone https://github.com/redpanda313/forgeheart.git
cd forgeheart
npm install
npm run dev
```

→ **http://localhost:5180**

```bash
npm run build    # production files in dist/
npm run preview  # smoke-test the build
```

## What’s in the game

- Workshop tutorial — wake Elias, fight rogue automata
- Sky Empire market training — harvest, craft, hire, expand your bay
- Mega-city scaffold — home district, grand market, sky routes (in progress)
- Game Maker mode — free-fly city editor
